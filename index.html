Privacy Policy

This Privacy Policy describes how Car Earth BD uses and protects any information when using our application.

Information We Access

Our application integrates with Facebook using the Facebook Graph API to publish vehicle listings to the official Car Earth BD Facebook Page.

The application may request the following permissions:

  • pages_manage_posts
  • pages_read_engagement
  • pages_show_list

Purpose of Data Usage

The data is used solely to automate posting of vehicle listings from our internal system to our Facebook page.

Data Storage

We do not collect, store, or share personal data of Facebook users.

Third-Party Sharing

No user data is sold or shared with third parties.
